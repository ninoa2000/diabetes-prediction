# Chronic Disease Prediction System - Type 2 Diabetes

A comprehensive web application for predicting Type 2 Diabetes using machine learning.

## Architecture
- **Frontend**: Vue 3 + Vite
- **Backend**: Spring Boot (Java)
- **Prediction API**: Flask (Python) + Scikit-learn/XGBoost

## How to Run
1. Ensure you have Node.js, Java 17+, and Python 3.9+ installed.
2. Run `start_all.bat` to launch all services.
3. Access the frontend at `http://localhost:3000`.

## Project Structure
- `/frontend`: Vue.js application.
- `/backend`: Spring Boot REST API.
- `/prediction`: Python Flask server for ML model inference.
